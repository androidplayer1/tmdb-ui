# Torrentie Configuration UI

Hosted at [androidplayer1.github.io/tmdb-ui](https://androidplayer1.github.io/tmdb-ui/)

## Features

- Select torrent providers
- Choose sorting mode (quality, seeders, size)
- Enable relevance-based sorting
- Configure excluded keywords and adult content filters
- RealDebrid / AllDebrid API key setup
- YTS and Jackett integration
- Install or download the configured plugin directly
